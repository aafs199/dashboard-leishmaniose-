# Dashboard Leishmaniose Visceral

Dashboard interativo para análise de dados epidemiológicos.

## Como usar:
1. Acesse o link online
2. Faça upload dos arquivos Excel
3. Visualize os gráficos

## Arquivos necessários:
- incidencialetalidadelv.xlsx
- casoshumanoslvregional.xlsx
- anual 2014-2023.xlsx

## Tecnologias:
- Python
- Streamlit
- Plotly
